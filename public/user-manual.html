系统概述

了解苏小智中枢智能体的核心能力与应用场景

苏小智中枢智能体是一个面向企业级的 AI 智能助手平台，集成了大语言模型对话、技能管理、MCP（Model Context Protocol）工具服务、数字警员（预设 AI 角色）等核心能力，旨在帮助用户高效完成日常办公、数据分析、文档处理等各类任务。

AI 智能对话
多模型接入、流式响应、文件上传分析，让 AI 成为您的工作伙伴
技能市场
上传和管理自定义 AI 技能包，扩展 AI 的专业领域能力
MCP 工具服务
接入标准化的工具调用协议，让 AI 能够执行实际操作
数字警员
预设 AI 角色，绑定专属技能与工具，即选即用
ⓘ
适用场景：日常办公辅助、数据分析报告、文档生成处理、专业知识问答、业务流程自动化等。系统通过浏览器访问，支持主流浏览器（Chrome、Edge、Firefox 等）。

快速开始

从登录到第一次对话，只需几步

第一步：通过统一登录平台进入系统

  1. 访问统一登录平台

    在浏览器中打开单位统一登录平台门户，完成身份认证。平台已集成零信任安全架构，确保访问链路安全可信。

  2. 点击进入"苏小智中枢智能体"

    在统一登录平台的应用列表中找到"苏小智中枢智能体"入口，点击后系统将自动携带您的身份凭证，以零信任方式免密登录进入系统。

  3. 自动进入主页

    认证通过后，系统自动跳转至"做一做"主页，无需输入任何账号密码，即可立即开始与 AI 对话。

ⓘ
说明：苏小智中枢智能体采用零信任安全架构，所有用户均通过统一登录平台完成身份认证后免密进入。新用户首次使用前，请确认您的账号已在系统中开通并通过审批。如无法在应用列表中找到入口，请联系系统管理员确认权限。

第二步：开始第一次对话

  1. 选择模型

    点击输入框上方的模型选择器，从可用模型列表中选择合适的大语言模型。选择后将自动保存，下次使用默认选中。

  2. 输入问题

    在底部输入框中输入您的问题或需求，支持普通文本和多行输入。按 Enter 发送，Shift + Enter 换行。

  3. 查看回复

    AI 将以流式方式实时返回答案。您可以查看思考过程（点击折叠面板展开）、复制回复内容、或重新生成答案。

第三步：了解导航结构

角色与权限

了解不同角色的职责与可访问的功能范围

系统采用四级角色体系，每个角色拥有不同的功能访问权限。管理员可以为用户分配一个或多个角色。

超级管理员
拥有系统全部功能权限。可访问所有页面，管理所有后台功能，执行两级审核（终审），管理系统配置和模型。
部门管理员
管理系统管理后台的各个子模块：账号审核、部门管理、角色分配，以及技能/警员/MCP 的一级审核。可进行统一授权管理。

专门负责安全审计，可查看登录日志、聊天日志和操作日志，对系统的安全使用情况进行监督。

使用 AI 智能问答、浏览技能市场、使用 MCP 服务和数字警员。可以上传自定义技能和注册 MCP 服务，经审核后发布。

AI 智能问答

系统核心功能 — 与 AI 进行多轮对话，完成各种任务

对话界面组成

左侧：会话列表
新建对话、查看历史记录、重命名/删除会话、"我的产物"文件管理
中间：对话区域
AI 流式回答展示、思考过程折叠面板、消息复制与重新生成
右侧：导航点
对话目录导航，快速跳转到任意问答位置，自动高亮当前位置
底部：输入区域
模型选择、文件上传、警员/MCP选择、@技能调用、发送/停止按钮

核心功能详解

⚙
模型选择
在对话输入框上方可以切换当前使用的 AI 模型。系统支持多种大语言模型，不同的模型在推理能力、响应速度和擅长领域上有所不同。选择后将保存在本地，下次登录自动沿用。
📎
文件上传与分析
支持上传 PDF、DOCX、XLSX、CSV、TXT、MD、JPG、PNG 等格式文件。可通过点击上传按钮、拖拽文件到输入区、或粘贴（Ctrl+V）方式上传。上传后 AI 将基于文件内容进行分析和回答。多个文件可同时上传，显示在输入框上方。
💡
思考过程可视化
AI 回答时，您可以展开折叠面板查看其"思考过程"——包括技术分析步骤、工具调用的参数与结果、处理文本等。这有助于了解 AI 的推理逻辑和结果来源。思考过程以时间线形式展示，支持逐条展开或折叠。
🔄
会话管理
左侧会话列表保存了所有的历史对话。支持：新建对话（开始全新话题）、重命名会话（长按或右键编辑标题）、删除会话（永久删除历史记录）。点击历史会话即可快速切换查看。系统还支持"我的产物"面板，集中管理您上传的文件和 AI 生成的输出文件。
✓
小技巧：给 AI 提供清晰、具体的指令会得到更好的回答。例如，"帮我写一份关于XX项目的周报，包含本周完成的工作、遇到的问题和下周计划"比"写一份周报"效果更好。

技能库

管理和使用 AI 技能包，扩展 AI 专业能力

技能（Skill）是预设的提示工程包，可以为 AI 注入特定领域的专业知识和处理逻辑。技能库分为两个标签页：

👤
我的技能
展示您自己创建和上传的所有技能。您可以在此：上传新技能（定义技能名称、分类、描述和提示词内容）、申请发布（提交审核后公开到技能市场）、申请下架（将已发布的技能撤回）、删除未发布的技能。技能状态包括：未发布、审核中、已发布、已驳回。
🌐
通用技能
浏览所有已公开发布的技能。可以通过分类筛选和关键词搜索来查找所需技能。发布的技能可供所有用户在对话中使用，提升特定领域任务的处理效果。
ⓘ
审核流程：技能从创建到公开发布需要经过两级审核——先由部门管理员初审，再由超级管理员终审。审核通过后，技能将出现在通用技能市场中供所有人使用。

MCP 专区

注册和管理 Model Context Protocol 工具服务

MCP（Model Context Protocol）是一种标准化的工具调用协议，允许 AI 模型通过工具服务器执行实际操作，如查询数据库、调用 API、操作文件等。

🔧
我的 MCP 服务

展示您自己注册的 MCP 服务。注册时需提供：

  • 名称和编码：服务的标识信息
  • 类型和描述：分类和功能说明
  • 服务器 URL：MCP 服务的连接地址
  • 配置 JSON：服务所需的参数配置
  • 图标：便于识别的服务图标
🌐
通用 MCP 服务
浏览所有已发布上线的 MCP 服务。点击服务卡片可查看详细信息，包括服务配置参数。已发布的服务可以在对话中被 AI 调用，扩展对话的能力边界。
⚠
注意：MCP 服务的发布和下架同样需要经过两级审核。请确保服务 URL 和配置参数正确无误后再提交审核。

数字警员库

创建和使用预设 AI 角色，提升专业场景下的对话效率

数字警员是预先配置好的 AI 角色，每个警员绑定了一套系统提示词、关联技能和 MCP 服务。在对话中选择警员后，AI 将自动加载其专属配置，以特定的人设和专业能力进行回答。

👤
我的警员
创建和管理自定义数字警员。创建时需配置：名称和描述（警员的基本信息）、系统提示词/头像（定义警员的角色定位和行为方式）、关联技能（为警员绑定专业能力）、关联 MCP 服务（为警员绑定工具能力）。警员支持按状态筛选（已发布、审核中、已下架），支持编辑和删除操作。
🌐
通用警员
浏览所有已公开发布的警员。看到合适的警员可以一键复制到"我的警员"列表中使用，无需从零创建。
✓
使用场景举例：创建一个"数据分析师"警员，绑定数据处理的技能和 Excel 操作的 MCP 服务，使其专注于数据分析和报表生成任务。

我的权限

查看个人角色、部门信息及已授权的资源清单

🔒
权限总览
在"我的权限"页面，您可以查看：当前角色和所属部门、可访问的页面列表数量、已授权的资源总数，以及具体的资源详情（技能、MCP 服务、数字警员的授权清单）。授权资源以标签形式展示，清晰显示每种资源的授权状态。
ⓘ
说明：如需获取更多资源访问权限，请联系部门管理员或超级管理员通过"统一授权"功能进行授权。

账号审核

管理员审核新用户的账号开通申请。用户列表支持按审批状态（待审核/已通过/已驳回）、角色和部门进行筛选，也支持按姓名、账号、手机号进行搜索。

操作步骤：在用户列表中找到待审核的用户，点击"通过"或"驳回"按钮。通过时需确认用户信息是否正确；驳回时需填写驳回原因。支持批量审批操作。管理员还可以编辑用户的基本信息（姓名、手机号、所属单位、部门、密码等）。

模型管理

仅超级管理员可访问。在此配置系统可用的 AI 大语言模型，包括：

  • 模型名称和标识符：在系统中显示的名称和唯一标识
  • API 地址和密钥：模型服务的连接信息和认证凭据
  • 超时设置：连接超时、读取超时和最大重试次数
  • 思考模式开关：控制是否启用模型的思考链展示
  • 管理员可见性：控制模型是否仅对管理员可见

部门管理

管理员可对组织部门进行增删改查操作。功能包括：

  • 创建部门：填写部门名称和编码
  • 编辑部门：修改部门信息
  • 删除部门：单个或批量删除
  • 用户配额管理：设置部门可容纳的用户数量上限
  • 成员管理：向部门添加成员或从部门移除成员
  • 搜索：按部门名称或编码快速查找

角色管理

管理员为用户分配系统角色。操作流程：先按部门筛选用户列表，选择目标用户，查看其当前拥有的角色，然后为其分配或移除角色。系统支持四种角色：超级管理员 部门管理员 安全审计员 普通用户。

审核系统（两级审核）

系统中技能、数字警员和 MCP 服务的发布、下架和删除操作都需要经过审核。采用两级审核机制确保内容质量：

一级审核：部门管理员
用户提交发布申请后，首先由所属部门管理员进行初审。部门管理员审核请求内容、检查配置是否正确、评估是否符合发布标准。可查看申请人信息、申请内容和申请时间。
二级审核：超级管理员
部门管理员审核通过后，申请进入超级管理员的终审队列。超级管理员进行最终审核，批准后资源才正式发布上线。两级审核状态都以标签形式展示，清晰可见。

审核页面顶部显示各状态的统计数量（全部、待审核、已通过、已驳回、已下架），方便管理员快速了解待处理工作量。每个待审核项的详细信息以卡片形式展示。

统一授权

管理员为特定用户或整个部门授予资源（技能、MCP 服务、数字警员）的访问权限。

🔍
授权列表
按用户或部门维度查看已有的授权记录。支持按部门和用户级联筛选。每条授权记录展示：资源类型（技能/MCP/警员）、资源名称、授权目标、授权人、授权状态和有效期。
➕
新增授权
  1. 选择授权目标类型：用户 或 部门
  2. 选择具体的用户或部门
  3. 选择资源类型：技能 / MCP 服务 / 数字警员
  4. 选择具体要授权的资源
  5. 设置授权有效期
  6. 确认提交
ⓘ
提示：可以随时在授权列表中撤销（取消）已授予的权限。撤销后目标用户/部门将无法再访问该资源。

日志查询（安全审计）

安全审计员和超级管理员可通过此功能查看和审计系统的使用情况。日志分为三个标签页：

🔑
登录日志
记录所有用户的登录行为，包含：账号、姓名、IP 地址、登录时间、登录方式（零信任免密）、登录状态。支持按关键词、角色和时间范围筛选。
💬
聊天日志
记录所有用户的对话内容，包含：用户姓名、角色、提问内容、AI 回答内容、对话时间、IP 地址。支持按关键词、角色和时间范围筛选，以分页表格展示。
📝
操作日志
记录所有管理员的系统操作，包含：操作用户、账号、角色、操作类型（创建/更新/删除/分配/认证/审核/申请）、操作模块、操作内容、IP 地址、操作时间。支持按操作类型和模块筛选，支持按日期范围查询，并提供导出 Excel 功能便于离线分析。

常见问题

使用过程中可能遇到的常见问题和解答

Q 如何登录苏小智中枢智能体？
A：请通过单位统一登录平台进入，在应用列表中找到"苏小智中枢智能体"并点击，系统将通过零信任方式自动完成免密登录。无需手动输入账号密码。
Q 在统一登录平台看不到苏小智的入口怎么办？
A：您的账号可能尚未开通使用权限。请联系您所在部门的管理员或超级管理员，为您开通账号并分配相应的角色权限。
Q AI 回答不准确怎么办？
A：您可以尝试：1）换一种方式重新描述问题；2）使用更具体、更详细的提示；3）上传相关文件作为参考资料；4）切换不同的 AI 模型重试；5）选择合适的数字警员以获得更专业的回答。
Q 上传的文件大小有限制吗？
A：系统对上传文件大小有一定限制，具体限制取决于服务器配置。建议上传的文件控制在合理大小范围内。过大的文件可能影响 AI 处理速度。
Q 如何分享我的技能或警员给其他人使用？
A：在"我的技能"或"我的警员"中点击"申请发布"，经两级审核通过后，您的资源将出现在通用市场中，所有用户都可以搜索和使用。
Q 如何切换亮色/暗色主题？
A：系统支持跟随系统的主题设置自动切换亮色/暗色模式。如需手动切换，请在系统设置中调整主题偏好。登录页面默认使用亮色主题。

提升使用效率的实用技巧和快捷操作

Enter 发送消息
在输入框中按 Enter 键直接发送消息，Shift + Enter 换行输入多行文本
Ctrl+V 粘贴文件
复制文件后，在对话页面按 Ctrl+V 可直接将文件上传为对话附件
拖拽上传文件
将文件从桌面或文件夹直接拖拽到对话输入区域，即可自动上传为附件
停止生成
AI 回答过程中点击停止按钮，可随时中断生成，已生成的内容会保留
重新生成回答
对 AI 的回答不满意？点击消息旁的重试按钮，用相同问题重新生成答案
复制回复内容
每条 AI 回复旁都有复制按钮，一键将回答内容（含格式）复制到剪贴板
长按重命名
在左侧会话列表中，长按历史会话条目可触发重命名操作
回到最底部
当向上滚动查看历史消息时，右下角会出现"回到底部"按钮，点击即可跳转
